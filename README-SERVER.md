# 서버 실행 가이드

## 일반 실행
```bash
npm run dev
```

## 자동 재시작 서버 실행
서버가 종료되면 자동으로 재시작하는 스크립트:

```bash
./dev-server.sh
```

## 포트 확인 및 종료
```bash
# 5175 포트 사용 중인 프로세스 확인
lsof -i:5175

# 5175 포트 프로세스 종료
lsof -ti:5175 | xargs kill -9
```

## 접속 주소

- 네트워크: http://192.168.0.184:5175/
